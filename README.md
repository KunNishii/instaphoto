# README

## Title
## instaphoto

## Features
#### instagramのクローンアプリ
#### 作成の目的：ruby,ruby on Railsの学習のため
 
## Requirement
* ruby 2.6.5
 
## Installation
### ローカルサーバーを立ち上げ、localhost:3000へアクセスしてください。
 
## Usage
#### ログイン画面よりログインを行なってください。ログイン中は画像のupが可能です。ログインしていなくても画像の閲覧は可能です。

### 実装機能
* ログイン・ログアウト機能
* 画像up,削除機能
* いいね機能
* follow機能
